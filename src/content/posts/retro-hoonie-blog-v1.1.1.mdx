## Not Found 페이지 추가

> **"길을 잃으셨나요? 이제 더 쉽게 귀환할 수 있습니다!"**

### 변경 내용

기존의 Next.js 기본 404 오류 페이지를 블로그 테마에 어울리는 404 페이지로 교체했습니다.

### 주요 개선사항

- 🔍 사용자 친화적인 오류 메시지로 상황 파악이 쉬워졌습니다
- 🏠 홈으로 빠르게 돌아갈 수 있는 귀환 포탈(버튼)이 추가되었습니다
- 🌓 라이트/다크 모드 모두 지원합니다
- 🔊 접근성 개선: 스크린 리더와 호환되는 ARIA 레이블 적용

### 관련 작업

- Issue: [#10](https://github.com/starcradle101/hoonie-blog/issues/10)
- PR: [#11](https://github.com/starcradle101/hoonie-blog/pull/11)

## 테마 쿠키를 세션에서 지속으로 변경

> **"이제 여러분의 선호 테마가 기억됩니다!"**

### 변경 내용

브라우저 세션 종료 후에도 여러분이 선택한 테마 설정이 유지됩니다.

### 주요 개선사항

- 💾 테마 설정이 7일간 지속되어 매번 다시 설정할 필요가 없습니다
- 🔒 `SameSite=Lax` 속성 추가로 보안이 강화되었습니다
- 🔄 테마 변경 시 자동으로 설정이 저장됩니다

이제 매번 테마를 다시 설정하느라 시간을 낭비하지 마세요! 여러분의 취향을 기억하고 방문할 때마다 적용해 드립니다.

### 관련 작업

- Issue: [#12](https://github.com/starcradle101/hoonie-blog/issues/12)
- PR: [#13](https://github.com/starcradle101/hoonie-blog/pull/13)

자세한 내용은 [테마 쿠키 변경 회고](http://hoonieblog.xyz/blog/retro-cookie-theme-switch)에서 확인하세요!

## 프로젝트 내부 매직 넘버 처리

> **"눈에 보이지 않는 곳의 대대적인 정리!"**

### 변경 내용

게임 내부 코드를 재정비하여 더 안정적인 블로그 경험을 제공합니다.

### 주요 개선사항

- 📁 `src/constants` 디렉토리 신설로 데이터 관리 체계화
- 🗂️ 기능별 상수 파일 분리로 시스템 최적화:
  - `social.tsx` - 소셜 연결 관련 상수
  - `site.ts` - 사이트 기본 정보
  - `posts.ts` - 포스트 관련 설정
  - `images.ts` - 이미지 처리 상수
  - `date.ts` - 날짜 형식 설정
  - `theme.ts` - 테마 관련 상수
- 🛡️ TypeScript 타입 시스템 강화로 버그 발생 가능성 감소

이 변경은 눈에 보이지 않지만, 블로그의 안정성과 향후 업데이트 속도를 크게 향상시킵니다!
